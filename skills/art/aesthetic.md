# Visual Aesthetic System

**A warm, approachable aesthetic combining Excalidraw's hand-drawn charm with a cream/teal brand palette.**

This is the **default aesthetic** included with the art skill. Customize it or create your own in `aesthetics/`.

---

## Core Concept: Warm Expertise

Every visualization balances **professional clarity** with **human warmth** — clean compositions rendered with hand-drawn, sketchy energy that feels approachable and trustworthy.

**The Philosophy:** *"Expert advice from a friend."*
- Hand-drawn warmth meets professional clarity
- Color accents provide energy without overwhelming
- Sketchy lines convey thought-in-progress and approachability
- Simplicity = Clarity (remove everything that doesn't serve the message)

---

## The Look

### What We Want
- **Excalidraw-style hand-drawn lines** — Sketchy, rough, whiteboard aesthetic
- **Light cream backgrounds** — Warm, readable, inviting
- **Teal expertise accents** — Trust, expertise, credibility
- **Burnt orange warmth accents** — Energy, approachability, action
- **Minimal compositions** — 2-4 elements maximum, plenty of breathing room
- **Subtle shadows** — Soft, warm, not harsh

### Reference Styles
- **Excalidraw** — Hand-drawn whiteboard diagrams with rough edges
- **Newsletter headers** — Clean, warm, editorial quality
- **Sketch noting** — Casual, hand-drawn information graphics
- **Anthropic editorial** — Flat colors, conceptual metaphors

### What to AVOID
- Over-polished corporate vectors
- Excessive color saturation
- Perfect geometric shapes (keep it sketchy)
- Cluttered compositions
- Harsh shadows or 3D effects
- Photorealistic elements
- Dark/moody backgrounds
- **Using "Excalidraw" in prompts** — The word renders as literal text in images. Use "hand-drawn sketch style" or "whiteboard aesthetic" instead

4. **Cream backgrounds** — provide warmth and readability
5. **Highlights** — subtle, soft shadows only

## Linework Characteristics

The sketchy hand-drawn quality is critical:

1. **Rough, imperfect strokes** — Like drawing on a whiteboard quickly
2. **Variable line weight** — Thicker at connections, thinner at ends
3. **Wobbly curves** — No perfect circles or smooth Beziers
4. **Multiple overlapping strokes** — Looks hand-drawn, not vector
5. **Gaps and breaks** — Lines don't always connect perfectly
6. **Sketchy hatching** — For shading or emphasis areas
7. **Casual confidence** — Quick strokes, not labored perfection

### This is NOT
- Clean vector illustration
- Perfectly smooth curves
- Uniform stroke weight
- Polished corporate graphics

---

## Composition Rules

1. **Minimal elements** — 2-4 key components maximum
2. **Generous negative space** — 40-50% breathing room
3. **Balanced placement** — Professional but not rigid
4. **Clear hierarchy** — Clear focal point, supporting elements
5. **Warm depth** — Subtle shadows create depth without harshness
6. **Hand-drawn feel** — Even geometric shapes look sketched

### Layout Strategies
- **Whiteboard sketch** — Casual hand-drawn explanation aesthetic
- **Editorial illustration** — Clean conceptual metaphor
- **Floating elements** — Components on cream background with soft shadows
- **Information graphics** — Data visualization with warmth

## Visual Metaphors

**How to translate concepts into this aesthetic:**

| Concept Type | Visual Strategy | Style Notes |
|--------------|-----------------|-------------|
| System Architecture | Connected boxes with teal pathways | Sketch + teal connections |
| Data Flow | Hand-drawn arrows with orange highlights | Orange flow direction, charcoal sketch |
| Process Steps | Numbered circles with connecting lines | Sketchy circles, teal highlights |
| Hierarchies | Tree diagrams with warm accents | Charcoal branches, teal/orange highlights |
| Relationships | Network graphs with colored nodes | Teal nodes, charcoal sketch edges |
| Transformations | Before/after with orange transition | Orange highlights show transformation |

**The Formula:**
```
1. Identify core concept (single clear idea)
2. Choose appropriate metaphor (diagram type)
3. Sketch with rough charcoal lines (Excalidraw feel)
4. Add strategic accents (teal expertise, orange warmth)
5. Apply subtle soft shadows (warm depth)
6. Place on cream background (inviting readability)
7. Leave generous negative space (clarity and calm)
```

- Central node with teal highlight
- Radiating branches (charcoal sketch)

## Typography (When Needed)

**Hand-drawn casual script:**
- Looks written by hand, not typed
- Slightly irregular baseline
- Variable character spacing
- Uppercase for emphasis, lowercase for body
- Charcoal or teal, depending on emphasis

**Avoid:**
- Perfect geometric fonts
- All-caps technical fonts
- Serif fonts
- Overly decorative fonts

## Base Prompt Prefix (Consistency Lock)

**Every image generation for this aesthetic must be prepended with this base prefix.** This ensures visual consistency across all illustrations in a set. The prefix locks down the parameters that, if varied, break cohesion.

```
Hand-drawn editorial illustration in rough sketch style on warm cream paper background. Medium-weight charcoal ink pen lines with wobbly hand-drawn quality, multiple overlapping strokes like whiteboard markers. Variable line weight — thicker at connections, thinner at ends. Soft warm shadows, not harsh. Illustration fills 60 percent of frame with 40 percent cream background breathing room. Muted color palette: charcoal sketch lines dominate at 70 percent, deep teal accent at 10-15 percent for expertise elements, burnt orange accent at 5-10 percent for warmth and action elements.
```

### Consistency Lock Parameters

| Parameter | Locked Value | Why |
|-----------|-------------|-----|
| Line weight | Medium charcoal ink pen | Too thin = pencil, too thick = marker/cartoon |
| Line quality | Wobbly, overlapping strokes | Consistent Excalidraw-like imperfection |
| Fill ratio | 60% illustration, 40% cream | Prevents overcrowding, editorial feel |
| Color balance | 70% charcoal, 15% teal, 10% orange, 5% shadow | Consistent palette weight |
| Shadow intensity | Soft, warm, subtle | Not dramatic, not absent |
| Composition | 2-4 elements maximum | Minimal and clear |

**When creating project-specific aesthetics**, derive a project-specific prefix from this base. Override only what needs to change (camera angle, background color, text treatment) and lock the overrides in the project's aesthetic file.
